# 🛍️ Bandage E-Commerce Frontend

A modern, responsive, and scalable e-commerce frontend application built with React.
Designed with a focus on clean UI/UX, performance, and reusable component architecture.

---

## 🚀 Live Demo

👉 https://ecommerce-rouge-xi-37.vercel.app/

---

## ✨ Features

* ⚡ Fast development with Vite
* 🎨 Pixel-perfect UI using Tailwind CSS
* 📱 Fully responsive design (mobile-first)
* 🧭 Client-side routing with React Router (v5)
* 🧩 Modular and reusable component structure

### UI Sections Implemented

* TopBar (contact & social)
* Header (navigation)
* Hero Section
* Featured Categories
* Bestseller Products Grid
* Featured Content Section
* Services Section
* Featured Posts (Blog Preview)
* Brand Logos Section
* Footer

---

## 🏗️ Tech Stack

**Frontend**

* React (Vite)
* Tailwind CSS
* React Router DOM (v5)
* React Icons / Lucide React

**State Management**

* Redux (basic setup ready)
